First-class structures

				// User = { name }
				// UserInfo = (User) => UI
				function UserInfo(user) {
				    return { type: 'label', text: user.name };
				}

UI composition

				function Header(user) {
				    return { type: 'div', children: [UserInfo(user)] };
				}
			
Header(someUserInfo);

Friendly syntax

				function Header(user) {
				    return (<div>
				        <UserInfo user={user} />
				    </div>);
				}
			
<Header user={someUserInfo} />

Templates

  • Always consist of a big amount of elements
  • Implicitly depend on some context
  • Are not reusable
  • Have additional syntax

Components as functions or first-class UI

  • As small as possible
  • Explicitly depend on data which they receives
  • Full JS support and all possibilites of Functional Programming
  • Reusable by default
